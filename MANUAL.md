# 📘 คู่มือการใช้งาน U-ra Seafood

---

## สำหรับลูกค้า (ผู้ใช้ทั่วไป)

### 1. สมัครสมาชิก

1. เข้าไปที่ `http://localhost:3000`
2. กด **"สมัครสมาชิก"** ที่มุมขวาบน
3. กรอก: **ชื่อ-นามสกุล**, **อีเมล**, **รหัสผ่าน** (อย่างน้อย 6 ตัว)
4. กด **"สร้างบัญชี"**

---

### 2. เข้าสู่ระบบ

1. กด **"เข้าสู่ระบบ"** ที่มุมขวาบน
2. ใส่ **อีเมล** และ **รหัสผ่าน**
3. กด **"เข้าสู่ระบบ"**

---

### 3. การสั่งซื้อสินค้า

1. เลือกสินค้าในหน้าแรก → กด **"+ ใส่ตะกร้า"**
2. ปรับจำนวนสินค้าในตะกร้าด้านขวา
3. กรอก **เบอร์โทรศัพท์** (สำหรับพนักงานติดต่อ)
4. ต้องการใช้แต้ม → ใส่จำนวนแต้มในช่อง "ใช้แต้มสะสม"
5. กด **"✅ ยืนยันสั่งอาหาร"**

> 💡 ทุก 10 บาท จะได้ 1 แต้มสะสม สามารถแลก 1 แต้ม = ลด 1 บาท

---

### 4. ดูประวัติออเดอร์

1. คลิกชื่อตัวเอง (👤) หรือ ⭐ แต้ม ที่ Navbar
2. เปิดแท็บ **"🛒 ประวัติสั่งซื้อ"**
3. กด **"▼ ดูรายการ"** เพื่อดูรายการสินค้าในออเดอร์นั้น

| สีสถานะ | ความหมาย |
|---------|----------|
| 🟡 ⏳ รอยืนยัน | ออเดอร์ถูกรับแล้ว รอ admin ยืนยัน |
| 🟣 💳 ชำระแล้ว | admin ยืนยันรับชำระเงินแล้ว |
| 🟢 🚚 จัดส่งแล้ว | สินค้าถูกจัดส่งแล้ว |
| 🔴 ❌ ยกเลิก | ออเดอร์ถูกยกเลิก |

---

### 5. ชำระเงิน (แนบสลิป)

1. ไปที่ `/checkout` หรือ กด **"💳 ชำระเงิน"** ใน Navbar
2. โอนเงินตามยอดที่แสดง
3. กด **"อัปโหลดสลิป"** แนบรูปสลิปการโอน
4. กด **"ยืนยันการชำระเงิน"**

---

### 6. รีวิวสินค้า

1. เข้าสู่ระบบก่อน
2. ดูสินค้าในหน้าแรก → คลิก **"⭐ เขียนรีวิวแรก"** หรือ **"รีวิว"** ใต้สินค้า
3. เลือกดาว 1-5 ⭐
4. ใส่ความคิดเห็น (ไม่บังคับ)
5. กด **"✅ บันทึกรีวิว"**

---

### 7. จัดการโปรไฟล์

เข้าที่ `/profile` แล้วเลือกแท็บ:

| แท็บ | ทำอะไรได้ |
|------|----------|
| 👤 ข้อมูลส่วนตัว | แก้ชื่อ, เบอร์, เปลี่ยนรหัสผ่าน |
| 📍 ที่อยู่จัดส่ง | เพิ่ม/แก้ไข/ลบ/ตั้งหลัก |
| 🛒 ประวัติสั่งซื้อ | ดูสถานะ, สั่งซ้ำ |
| ⭐ รายการโปรด | สินค้าที่กดหัวใจไว้ |
| 🔔 การแจ้งเตือน | เปิด/ปิดแจ้งเตือนโปรโมชัน |

## สำหรับผู้ดูแลระบบ (Admin)

### 1. เข้าสู่ระบบ Admin

1. ไปที่ `http://localhost:3000/admin/login`
2. ใส่ **Username** และ **Password** จาก `.env`
3. กด **"เข้าสู่ Admin Panel"**

---

### 2. หน้า Dashboard

URL: `/admin/dashboard`

แสดง:
- 📋 **จำนวนออเดอร์** แยกตามสถานะ
- 💰 **ยอดขายรวม** (ไม่รวมที่ยกเลิก)
- 📈 **กราฟยอดขายรายวัน** (7 วันล่าสุด)
- 🏆 **Top 5 สินค้าขายดี**
- 🕐 **ออเดอร์ล่าสุด 5 รายการ**

---

### 3. จัดการสินค้า

URL: `/admin`

#### เพิ่มสินค้าใหม่
1. กรอก **ชื่อสินค้า**, **ราคา (บาท)**, **สต็อก (ชิ้น)**
2. อัปโหลด **รูปสินค้า** (ลากวางหรือคลิกเลือก)
3. กด **"+ เพิ่มเมนู"**

#### แก้ไขสินค้า
1. กด ✏️ ที่สินค้า → ระบบจะไปหน้า Edit
2. แก้ไขข้อมูลที่ต้องการ
3. กด **"💾 บันทึก"** → ระบบจะกลับมาหน้าสินค้า

#### ลบสินค้า
1. กด 🗑️ ที่สินค้า
2. ยืนยันการลบในกล่อง confirm

#### + สั่งให้ลูกค้า (Admin Order)
1. กด **"+ ใส่ตะกร้า"** บนสินค้าที่ต้องการ
2. ระบุ **เบอร์โทรลูกค้า** → ระบบจะค้นหา user และสะสมแต้มให้
   - **ถ้าไม่ใส่เบอร์** → บันทึกเป็น "ลูกค้าทั่วไป" ไม่สะสมแต้ม
3. กด **"✅ ยืนยันสั่งอาหาร"**

---

### 4. จัดการออเดอร์

URL: `/admin/orders`

#### เปลี่ยนสถานะออเดอร์
แต่ละออเดอร์จะมีปุ่มเปลี่ยนสถานะ:

```
PENDING (รอยืนยัน)
   ↓ กด "✅ ยืนยัน"
PAID (ชำระแล้ว)
   ↓ กด "🚚 จัดส่ง"
SHIPPED (จัดส่งแล้ว)
```

#### ยกเลิกออเดอร์
- กด **"❌ ยกเลิก"** → ระบบจะ **คืนสต็อกสินค้าอัตโนมัติ**

#### ดูรายละเอียดออเดอร์
- กด **"ดูรายการ"** ใต้ออเดอร์ → เห็นรายการสินค้าย่อย, ข้อมูลลูกค้า, สลิปโอนเงิน

---

### 5. ตั้งค่า Discord Webhook

1. เปิด Discord Server → ช่อง → ⚙️ → Integrations → Webhooks
2. กด "New Webhook" → Copy URL
3. เปิดไฟล์ `.env` ใส่ URL:
   ```
   DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...
   ```
4. Restart server (`npm run dev`)

เมื่อมีออเดอร์ใหม่จะได้รับข้อความใน Discord ทันที 🔔

## ❓ ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|-------|--------|
| ไม่สามารถสร้างออเดอร์ได้ | เช็คว่า XAMPP (MySQL) รันอยู่ |
| รูปสินค้าไม่แสดง | เช็คโฟลเดอร์ `public/uploads/` มีสิทธิ์เขียนไฟล์ |
| สถานะออเดอร์ไม่อัปเดต | กด Refresh หน้าโปรไฟล์ |
| Discord ไม่ได้รับแจ้งเตือน | เช็ค `DISCORD_WEBHOOK_URL` ใน `.env` ถูกต้อง |
| ล็อกอิน Admin ไม่ได้ | เช็ค `ADMIN_USERNAME` และ `ADMIN_PASSWORD` ใน `.env` |
